# Негативные тесты

- Ввод:
    - Целое:
        1. Буквы
        2. Пустая строка
        3. +
        4. -
        5. Длина > 40
        6. Точка в числе
    
    - Вещественное:
        7. Буквы
        8. Пустая строка
        9. +
        10. -
        11. Длина мантиссы > 30
        12. Экспонента > 99999
        13. Экспонента < -99999
        14. Неверная форма вещественного числа

- Вычисления:
    15. Экспонента > 99999

        12. Число с точкой без эксп. с +
